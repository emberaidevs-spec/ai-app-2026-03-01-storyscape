> StoryScape is an AI-powered story and poem generator that creates unique and engaging content based on user input. It can generate short stories, poems, or even entire scripts, making it a useful tool for writers, students, and anyone looking for creative inspiration.

## ✨ Features

- Customizable story/poem length
- Various genres and themes
- User input for personalized content

## 🛠 Tech Stack

- Frontend: Pure HTML / CSS / JavaScript
- Backend: Vercel Serverless Functions (Node.js)
- AI: Groq API (Llama 3.3 70B)
- Groq AI API via Vercel serverless function + Web Speech API for voice input
- Dark theme with glassmorphism UI
- Fully responsive (mobile + desktop)
